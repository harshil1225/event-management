# 🎉 Event Management System

## 📖 Description
The Event Management System is a web-based application designed to simplify the process of planning, organizing, and managing events. It allows users to book events, select venues, manage catering services, and handle payments efficiently.

This system is useful for clients, event organizers, and administrators to manage event-related activities in a structured way.

---

## 🎯 Features

### 👤 User Management
- User Registration and Login  
- Update Profile  
- Change Password  
- Forgot Password  

### 📅 Event Management
- Add Event  
- Manage Event  
- Book Event  
- Search Event  
- Delete Event  

### 📍 Venue Management
- View available venues  
- Select venue  
- Allocate venue  

### 🍽️ Catering System
- Menu selection  
- Food ordering  

### 💳 Payment System
- Online Payment (UPI / Net Banking)  
- Offline Payment (Cash / Cheque)  

### 📝 Blog System
- Manage blogs  
- Rate blogs  

### ⭐ Feedback System
- Add feedback  
- View feedback  
- Update feedback  
- Delete feedback
